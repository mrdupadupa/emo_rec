# emo_rec
fun project to make a demo for the Long-night-of Science 
in the early development stage


## how to run:

`cd emo_rec/space-3d/src`

`npm start`

please use venvs

`pip install -r requirements`
`source name_of_venv/bin/activate`

`cd emo_rec/`

run the script:

`python emo_rec+sound.py`
